API Reference

Complete documentation for the YT Transcript AI API endpoints.

Analyze Endpoint

This is the main endpoint for analyzing YouTube videos.

HTTP Request
POST https://api.yttranscript.ai/v1/analyze
Request Parameters
Parameter Type Required Description
video_url String Yes Full YouTube video URL
language String No Preferred language code (default: auto-detect)
summarize Boolean No Whether to generate a summary (default: true)
extract_keywords Boolean No Whether to extract keywords (default: true)
Example Response
{
  "status": "success",
  "data": {
    "video_id": "XXXXXXXXXXX",
    "title": "Sample YouTube Video",
    "duration": "12:34",
    "transcript": [...],
    "summary": "This video discusses...",
    "keywords": ["keyword1", "keyword2"],
    "timestamp": "2023-06-15T12:00:00Z"
  }
}
Warning: Always include your API key in the Authorization header. Exposing it in client-side code may lead to unauthorized usage.

Frequently Asked Questions

Our AI achieves approximately 95% accuracy for videos with clear speech. Accuracy may vary based on audio quality, background noise, and speaker accents. For best results, use videos with official captions.

The free tier supports videos up to 30 minutes. Paid plans support videos up to 4 hours. For longer videos, consider using our chunk processing feature.

The free tier allows 5 requests per minute. If you exceed this limit, you'll receive a 429 status code. Implement exponential backoff in your code or upgrade your plan for higher limits.

Currently, we only support public YouTube videos. Support for unlisted videos is planned for Q4 2023.

Troubleshooting

Common Issues and Solutions
Issue Possible Cause Solution
401 Unauthorized Invalid or missing API key Verify your API key is correct and included in the Authorization header
404 Not Found Invalid YouTube URL Ensure the video exists and the URL is properly formatted
500 Server Error Temporary service issue Wait a few minutes and retry. Check our status page for updates
Empty transcript Video has no captions Try a different video or enable auto-captions on YouTube
Tip: For persistent issues, contact our support team with the request ID included in all API responses.

Code Examples

JavaScript Example
const analyzeVideo = async (videoUrl) => {
  try {
    const response = await fetch('https://api.yttranscript.ai/v1/analyze', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ video_url: videoUrl })
    });

    const data = await response.json();
    console.log('Video Summary:', data.data.summary);
  } catch (error) {
    console.error('Error:', error);
  }
};

analyzeVideo('https://youtube.com/watch?v=XXXXXXXXXXX');
Python Example
import requests

def analyze_video(video_url):
  url = "https://api.yttranscript.ai/v1/analyze"
  headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  }
  data = {"video_url": video_url}

  response = requests.post(url, headers=headers, json=data)
  if response.status_code == 200:
    return response.json()
  else:
    raise Exception(f"API request failed: {response.text}")

# Usage
result = analyze_video("https://youtube.com/watch?v=XXXXXXXXXXX")
print("Video Title:", result["data"]["title"])
